# Trace Vinculo Societario em Saude - Matriz Juridico-Funcional

Matriz conservadora para controle externo. Organiza fatos, normas e perguntas de apuracao sem declarar ilegalidade automatica.

## Regra

- So entra aqui caso com fonte primaria de saude, contrato publico e carga concomitante documentada.
- Os marcos `>=60h` e `>=80h` sao criterios tecnicos de triagem, nao conclusao juridica.
- A matriz aponta o que precisa ser confrontado com regime funcional e norma aplicavel.
